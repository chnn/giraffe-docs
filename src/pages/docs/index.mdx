# Documentation

Giraffe is a fast and flexible data visualization library for React.

## Features

- A high-level, but flexible API for specifying a wide variety of visualizations with a few simple concepts
- Accepts [tabular data](/docs/guides/table) via a table interface inspired by the theory of [Tidy Data](https://vita.had.co.nz/papers/tidy-data.pdf), paired with adapters for transforming input from a [variety of sources](/docs/guides/adapters)
- Uses a [columnar](https://observablehq.com/@mbostock/manipulating-flat-arrays) storage format for fast visualizations and interopability with WebWorkers
- Supports for grouping data by multiple variables, and encoding groups aesthetically in visualizations
- [First-class support for TypeScript](/docs/guides/typescript)
- [Easily extensible](/docs/guides/custom-layers) via a React-based API
